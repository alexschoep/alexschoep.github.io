Google's Deep Dream software uses convolutional neural networks to process images and create interesting effects. "Deep" refers to deep learning which is one of the fields that CNN's come from. "Dream" comes from the strange images that it produces. Instead of using the network to classify some object in an image, Deep Dream edits the image to maximize the number of that object in the image. The following links contain some more information.

Deep Dream - Wikipedia
github.com/deepdream

For example, if Deep Dream is told to look for eyes, anything in an image that has similar characteristics to an eye will have those characteristics maximized. The following image demonstrates this well.
